CODE::BLOCKS

Code::Blocks is a free C, C++ and Fortran IDE built to meet the most demanding needs of its users. It is designed to be very extensible and fully configurable. Finally, an IDE with all the features you need, having a consistent look, feel and operation across platforms.
Built around a plugin framework, Code::Blocks can be extended with plugins. Any kind of functionality can be added by installing/coding a plugin. For instance, compiling and debugging functionality is already provided by plugins!
Special credits go to darmar for his great work on the FortranProject plugin, bundled since release 13.12. We hope you enjoy using Code::Blocks!

Brackets is a source code editor with a primary focus on web development. Abbreviated as VBE, Visual Basic Editor is the tool used to create, modify and maintain Visual Basic for Applications (VBA) procedures and modules in Microsoft Office applications.
